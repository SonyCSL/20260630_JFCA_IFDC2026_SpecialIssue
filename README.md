# Chao1 Intensity Supplementary Data

This repository contains figure-level supplementary materials for the Chao1 intensity analysis of LC-MS profiles from synecoculture and conventional tea samples.

The public repository currently provides `Data 10`, which contains PowerPoint figure decks and a figure-deck manifest.

## Data Availability

Figure decks and figure-level supplementary materials are provided in Data 10. Large intermediate analysis outputs corresponding to Data 2-9 are not publicly deposited because they are high-volume intermediate computational products with limited standalone interpretability without the full analysis environment. These materials are available from the corresponding author upon reasonable request at syneco-comm@csl.sony.co.jp.

| Material | Public availability |
| --- | --- |
| Data 2-9 | Available from the corresponding author upon reasonable request at syneco-comm@csl.sony.co.jp. |
| Data 10 | Provided in this repository as PowerPoint figure decks and a figure-deck manifest. |

Each `SHA256SUMS.txt` file provides SHA-256 checksums for the files in the corresponding directory.
