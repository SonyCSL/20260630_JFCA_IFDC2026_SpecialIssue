# Chao1 Intensity Supplementary Data

This repository contains supplementary data files and data manifests for the Chao1 intensity analysis of LC-MS profiles from synecoculture and conventional tea samples.

The directory `Supplementary_Materials/` is organized by Supplementary Data number (`Data 2` to `Data 10`) to match the Methods and workflow diagram. `Data 1` is not included in this repository because it refers to the original upstream bancha tea source data before the present processing workflow.

Small files are stored directly in GitHub. Files larger than 100 MB are not stored in the repository; their locations are listed in the relevant manifest files as S3 object URIs.

Each `SHA256SUMS.txt` file provides SHA-256 checksums for the files in the corresponding directory.
