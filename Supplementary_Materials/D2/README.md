# Supplementary Data D2

Detected molecular formulae and per-sample LC-MS intensity values.

## File

- `D2_detected_formulae_per_sample_intensities.csv`

## Description

This table lists formula-level LC-MS features with intensity values for the 13 analytical profiles used in the analysis. The columns include the formula identifier, yearly Synecoculture and conventional tea profile intensities, KEGG identifiers where available, and the PC4 loading used in the upstream processing context.

This file is the formula-level intensity table from which the integrated D3 matrix is represented.
