# Supplementary Data 3

Integrated formula-level intensity matrix.

## File

- `Data_3_integrated_intensity_matrix_541x13.csv`

## Description

This matrix contains 541 formula-level features across 13 LC-MS analytical profiles. It is the integrated intensity matrix used as the fixed input for downstream intensity-cutoff, relative-abundance, Monte Carlo, diversity-estimation, and comparison analyses.

The table contains the formula identifier and the 13 analytical-profile intensity columns. Annotation identifiers and PCA diagnostic columns are not included in Data 3.
