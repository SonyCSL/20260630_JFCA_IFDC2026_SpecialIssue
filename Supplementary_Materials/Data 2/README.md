# Supplementary Data 2

Detected molecular formulae and per-sample LC-MS intensity values.

## File

- `Data_2_detected_formulae_per_sample_intensities.csv`

## Description

This table lists formula-level LC-MS features with intensity values for the 13 analytical profiles used in the analysis. The columns include the formula identifier, yearly Synecoculture and conventional tea profile intensities, and KEGG identifiers where available.

This file is the formula-level intensity table from which the integrated Data 3 matrix is represented.
