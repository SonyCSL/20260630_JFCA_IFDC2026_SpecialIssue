# Supplementary Data D5

Standard sample coverage setting and run manifests.

## Files

- `D5_sweep_sc_manifest_v36_9.csv`
- `D5_cutoff_manifest_v36_9.csv`
- `D5_mode_tag_manifest_v36_9.csv`
- `D5_run_metadata_v36_9_q2fast_reference.json`

## Description

D5 records the standard sample coverage and run-configuration information used by the downstream q=2 analysis. The included files document the coverage grid, signed cutoff definitions, comparison-mode labels, and run metadata for the reference q=2 workflow.

The full per-draw SSC calibration simulation log is not included in this repository snapshot. The deposited D5 files therefore document the retained SSC setting and the manifests required to identify the downstream run configuration.

File checksums are provided in `SHA256SUMS.txt`.
