# Supplementary Materials

This directory contains Supplementary Data D2-D10 for the Chao1 intensity analysis.

## Directory Map

- `D2`: detected molecular formulae and per-sample LC-MS intensity values.
- `D3`: integrated formula-level intensity matrix used as the analysis input.
- `D4`: relative-abundance distributions after signed intensity cutoffs.
- `D5`: standard sample coverage setting and run manifests.
- `D6`: formula-to-domain and PM/SM subset mapping tables.
- `D7`: Stage 2 Hill-number estimate summaries.
- `D8`: Synecoculture versus conventional comparison summaries.
- `D9`: summary-mode and sensitivity-analysis manifests.
- `D10`: figure-output manifests and small figure-support tables.

Files larger than 100 MB are referenced through manifest files rather than stored directly in GitHub. Public external payloads are available through the CloudFront prefix:

`https://d38f5mdcvtp0z3.cloudfront.net/JFCA_IFDC2026_SpecialIssue/`
