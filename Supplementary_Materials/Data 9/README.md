# Supplementary Data 9

Summary modes and sensitivity-analysis records.

## Files

- `Data_9_manifest.tsv`: manifest of large Data 9 payloads stored outside GitHub.
- `manifests/`: small input and figure manifests included directly in GitHub.
- `SHA256SUMS.txt`: SHA-256 checksums for files stored in this directory.

## Description

Data 9 records the summary-mode and sensitivity-analysis materials associated with the q=2 workflow, including input manifests, figure manifests, run-scope descriptions, and links to large summary payloads.

The repository stores small manifests directly. Larger summary archives are referenced in `Data_9_manifest.tsv` and should be retrieved through the external download link below.

Intermediate run directories, shard directories, and full `comparison_outer` payloads are not included as the default Data 9 data package because they are much larger than the summary objects needed to identify and review the reported analyses.
